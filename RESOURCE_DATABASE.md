## Contributor Schema

When adding a new entry, use this exact column order:

```markdown
| Resource Name | Category | Language | License | URL | Last Verified |
```

Each submission should meet these requirements:

- Use the official resource name.
- Choose a concise category, such as `STEM`, `Literacy`, `Textbooks`, or `Teacher Training`.
- List the main language or write `Multilingual`.
- Record the license as precisely as possible.
- Use a stable HTTPS URL.
- Set `Last Verified` to the date you checked the URL and license in `YYYY-MM-DD` format.
